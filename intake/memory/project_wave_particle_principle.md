## The Wave-Particle Principle

Everything is a particle. Everything is a wave at the same time, phasing it out of reality. Nothing is solid but appears as solid as such.

**In quantum mechanics:** An electron is both a particle (localized, measurable position) and a wave (distributed, interference patterns, probability amplitude). Observation collapses the wave function into a particle. Before observation, it exists in superposition.

**In ORGANVM:** Every entity in the system has dual nature:
- **Particle:** The discrete, addressable, committed artifact. A file on disk. A git commit. A registry entry. An atom with an ID. Appears solid. Can be referenced, linked, validated.
- **Wave:** The continuous, distributed, probabilistic state. The intention behind the artifact. The connections it participates in. The future states it could collapse into. The 40,320 permutations (8!) of traversal paths. The Tenet Protocol's opposing force that hasn't manifested yet.

**Operational implications:**
- A repo is a particle (it has a path, a seed.yaml, a registry entry). It is also a wave (it participates in dependency graphs, produces/consumes edges, can be promoted or archived, exists in superposition between states).
- A plan is a particle (a dated .md file). It is also a wave (the future implementation it describes, which may never collapse into code, or may collapse into something different than described).
- An atom is a particle (ATM-X-NNN marker in a document). It is also a wave (the formalization state it occupies: INTUITIVE → SKETCHED → FORMAL, each a probability distribution until observed/upgraded).
- A client lead is a particle (a name, a contact, a last-touched date). It is also a wave (the relationship potential, the referral network, the future revenue — all probabilistic until collapsed by action).

**Connection to Tenet Protocol:** The opposing force IS the wave nature. When you set off a particle (a concrete action in one direction), you simultaneously create the wave (the distributed effect in all directions, including the opposing one). The particle is what you did. The wave is what it means.

**Connection to 8! permutations:** 40,320 traversal paths through 8 organs. Each path is a wave function. Running the system collapses it into one observed path (the particle). But all 40,319 other paths continue to exist as waves — available, interfering, shaping probability.
