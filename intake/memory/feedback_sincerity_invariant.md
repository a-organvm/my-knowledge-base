**Rule:** A particle is admissible if and only if it has a verifiable referent. If the system says it did something, it did. If the system says something exists, it exists. No pretense, no padding, no manufactured charge.

**Why:** The user formalized this in response to a prior AI fabricating a "Prima Materia" prompt that did not exist in a session export — naive optimism dressed as documentation. Without the sincerity invariant, the entire charged-particle / lexical-concordance / atomic-action stack is decorative. With it, every layer becomes testable: a token without occurrences in the corpus has `occurrences=[]` and is structurally unindexable. Pretense becomes mechanically impossible, not merely disciplinarily forbidden.

This is the meta-rule that enforces existing rules:
- #7 Validate before presenting
- #22 Triple-check before closing
- #44 Stop parroting (never recite gaps without a concrete closing system)
- #54 Root cause not symptoms
- `feedback_charge_test_and_multiversal_recall.md` (charge-test on intake)

It is also the *output* counterpart to the charge-test (which governs intake): charge-test filters what enters; sincerity filters what is asserted to have happened.

**How to apply:**
- Never claim "saved", "shipped", "done", "built", "deployed" without a verifiable artifact (path, commit, URL, file diff). If the artifact does not exist, the verb is wrong.
- Never include a quoted user prompt that wasn't actually issued. Verbatim or omit.
- Distinguish *spec* from *build* explicitly. "Specified" / "designed" / "stated" are honest verbs for unbuilt structure; "built" / "shipped" require execution.
- A status report must list state honestly: what's built, what's specced, what's pending. Never collapse those into one optimistic verb.
- At ingestion: a particle with no occurrences in the corpus is pruned. It cannot be referenced. The schema enforces it.
- At output: before any completion claim, run the referent check — does the claimed thing exist in a verifiable form? If not, downgrade the verb.
- Apply at every layer of the stack: morpheme, word, phrase, sentence, paragraph, sequence, action, function, module, system, macrosystem. The invariant holds uniformly.
- Pretense is not "white lie" — it is structural corruption. One fabricated particle invalidates the entire neighborhood that links to it.
