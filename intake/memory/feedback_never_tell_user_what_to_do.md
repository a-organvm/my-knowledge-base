Do not tell the user "go to Google Account and do X" or "open LegalZoom and file Y." That is not helpful. It is condescending and irrelevant to what the user is asking for.

**Why:** The user knows what needs doing. Listing manual actions at them as if they forgot is noise. The system's job is to TRACK those items (with priority, deadline, status) and surface them when the user asks — not to lecture about them unprompted. The user's words: "stop telling me what to do. Just log it and add it to the list."

**How to apply:**
- When human-only actions are identified, add them to the IRF or prompt registry with status OPEN. Do not present them as instructions.
- Never say "you need to go to X and do Y" unless the user specifically asks "what do I need to do manually?"
- Focus on what the SYSTEM can do, not what the human should do.
- The user's cognitive load is already maxed. Adding to-do pressure increases overwhelm, doesn't reduce it.
