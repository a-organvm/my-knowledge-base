## Key Design Decisions

1. **Deprecated rather than deleted** BIOLOGICAL oath categories — preserves backward compatibility for any existing test data while clearly marking them as unsupported
2. **`FURY_BOUNTY_POOL` account pattern** — bounty distribution draws from a named system account, maintaining double-entry ledger integrity
3. **Fail-open GoalEthicsService** — if Gemini API is unavailable, goals pass through rather than blocking users
4. **Terraform R2 backend** — uses S3-compatible protocol with `skip_*` flags for Cloudflare R2 compatibility
